# Speedometer

This project is a simple speedometer for any vehicle. It displays the current speed.

## Components

- **Microcontroller**: ESP32 because is a powerful microcontroller with built-in Wi-Fi and Bluetooth capabilities.
- **Speed Display**: OLED controllable with IC2.
- **Speed Sensor**: Optical encoder, which is a type of sensor that detects the rotation of a wheel by counting the number of times a light beam is interrupted.
