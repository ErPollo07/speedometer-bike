# Speedometer

This project is a simple speedometer for any vehicle. It displays the current speed, as well as the maximum speed reached during the session.

## Components

- **Microcontroller**: An Arduino or similar microcontroller to read speed data and control the display.
- **Speed Display**: It has to be choose simple display compatible with the microcontroller (Arduino LCD)
- **Speed Sensor**: For this project a wheel encoder is better than a gps module, as it provides more accurate speed readings at low speeds too. And it's less expensive than a gps module.
