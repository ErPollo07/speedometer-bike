# Speedometer

This project is a simple speedometer for any vehicle. It displays the current speed.

## Components

- **Microcontroller**: An Arduino or similar microcontroller to read speed data and control the display.
- **Speed Display**: The display has to be OLED or LCD controllable with IC2.
- **Speed Sensor**: For this project a wheel encoder is better than a gps module, as it provides more accurate speed readings at low speeds too. And it's less expensive than a gps module.
  For the wheel encoder will be used a optical encoder, which is a type of sensor that detects the rotation of a wheel by counting the number of times a light beam is interrupted.
